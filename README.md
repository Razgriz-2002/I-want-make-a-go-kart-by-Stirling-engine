### 设计总览

1. 确定发动机的额定功率和运行的基本工况条件
2. 设计的主要部分
   1. 传动系统
   2. 热交换系统
3. 基本步骤
   1. 确定大概工况条件
      1. 冷热端运行温度
      2. 气体平均压力
      3. 发动机功率和转速
   2. 确定活塞半径和活塞活动范围，从而确定传动方式
   3. 设计内部热交换器（加热器，回热器，冷却器），热交换器参数，选择合适的循环体系方法
   4. 初步参数仿真和效率仿真，事物计算，反向调整发动机功率，追求输出功率和热交换器参数
